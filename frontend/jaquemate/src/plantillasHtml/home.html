Bienvenido a JaqueMate

JaqueMate es un proyecto de 2º año de DAW. Trata de un juego de ajedrez desarrollado con tecnologías web modernas. Es una combinacion de las asignaturas Desarrollo Web en entorno Cliente (DWEC) y Desarrollo Web en entorno Servidor (DWES)

Este proyecto combina la parte de backend y frontend para generar el ajedrez y darle funcionalidad. Se utilizan las librerias "chess.js" y "chessboard.js" para el apartado del tablero.
